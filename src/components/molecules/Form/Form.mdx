## Description

A form is a collection of interactive fields that enable users to input and submit data, with built-in validation for handling field errors.

## Tag

`<wa-form>`

## Props 

### buttonLabel
- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The label displayed on the submit button within the form.
