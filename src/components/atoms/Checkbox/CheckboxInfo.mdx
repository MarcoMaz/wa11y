# Checkbox
&nbsp;
<Story />
&nbsp;

## Description

A checkbox lets users choose multiple options from a list or indicate that a single item is selected.

## Tag

`<wa-checkbox>`

## ClassMap
- label
- input
- span

## Props 

### currentId

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The unique identifier for the checkbox element. Defaults to **default-id** if not provided.

### name

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The name attribute for the checkbox element. Defaults to **default-name** if not provided.

### checked

- **Type: Boolean**
- **Reflects to Attribute: true**
- **Description:** Indicates whether the checkbox is checked. Defaults to **false**.

### focused

- **Type: Boolean**
- **Reflects to Attribute: true**
- **Description:** Indicates whether the checkbox is focused. Defaults to **false**.

### label

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The label displayed next to the checkbox. Defaults to **default checkbox label** if not provided.
