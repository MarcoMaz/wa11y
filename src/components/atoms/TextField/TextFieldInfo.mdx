# TextField
&nbsp;
<Story />
&nbsp;

## Description

A text field enables users to input plain text values using a keyboard.

## Tag

`<wa-text-field>`

## ClassMap
- label
- input

## Props 

### currentId

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The unique identifier for the radio element. Defaults to **default-id** if not provided.

### name

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The name attribute for the radio element. Defaults to **default-name** if not provided.

### label

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The label displayed next to the radio. Defaults to **default radio label** if not provided.

### placeholder

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The text displayed in the text field when it is empty and not focused. If not provided, defaults to **'Search for the docs...'**.

### required

- **Type: Boolean**
- **Reflects to Attribute: true**
- **Description:** Default: false. When set to **true**, indicates that the input field must be filled out before submitting the form.
