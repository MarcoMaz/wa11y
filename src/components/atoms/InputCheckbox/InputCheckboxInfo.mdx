# InputCheckbox
&nbsp;
<Story />
&nbsp;

## Description

A input checkbox lets users choose multiple options from a list or indicate that a single item is selected.

## Tag

`<wa-input-checkbox>`

## ClassMap
- label
- input
- span

## Props 

### contentText

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The label displayed next to the input checkbox. Defaults to **default input checkbox content** if not provided.

### currentId

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The unique identifier for the input checkbox element. Defaults to **default-id** if not provided.

### name

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The name attribute for the input checkbox element. Defaults to **default-name** if not provided.

### checked

- **Type: Boolean**
- **Reflects to Attribute: true**
- **Description:** Indicates whether the input checkbox is checked. Defaults to **false**.

### focused

- **Type: Boolean**
- **Reflects to Attribute: true**
- **Description:** Indicates whether the input checkbox is focused. Defaults to **false**.
