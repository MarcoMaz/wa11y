# InputText
&nbsp;
<Story />
&nbsp;

## Description

A input text enables users to input plain text values using a keyboard.

## Tag

`<wa-input-text>`

## ClassMap
- label
- input

## Props 

### currentId

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The unique identifier for the input radio element. Defaults to **default-id** if not provided.

### name

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The name attribute for the input radio element. Defaults to **default-name** if not provided.

### label

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The label displayed next to the input radio. Defaults to **default input radio label** if not provided.

### placeholder

- **Type: String**
- **Reflects to Attribute: true**
- **Description:** The text displayed in the text field when it is empty and not focused. If not provided, defaults to **'Search for the docs...'**.

### required

- **Type: Boolean**
- **Reflects to Attribute: true**
- **Description:** Default: false. When set to **true**, indicates that the input field must be filled out before submitting the form.
